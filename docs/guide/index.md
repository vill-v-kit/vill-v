## 安装{#install}

::: tip 提示
vill-v 的前身为[@whitekite/utils](https://www.npmjs.com/package/@whitekite/utils)

现在 [@whitekite/utils](https://www.npmjs.com/package/@whitekite/utils) 只是作为vill-v的别名
:::

使用npm

```bash
$ npm i vill-v
```

使用yarn

```bash
$ yarn add vill-v
```

使用pnpm

```bash
$ pnpm add vill-v
```
