## 安装{#install}

:::tip 提示

vill-v 的前身为[@whitekite/utils](https://www.npmjs.com/package/@whitekite/utils)

现在 [@whitekite/utils](https://www.npmjs.com/package/@whitekite/utils) 只是作为 vill-v 的别名

:::

:::code-group

```bash [npm]
$ npm i vill-v
```

```bash [yarn]
$ yarn add vill-v
```

```bash [pnpm]
$ pnpm add vill-v
```

:::
